**Aurall Website**

Official website project for Aurall — an AI-powered platform for intelligent interaction recording and secure multimodal capture.
This repository contains the full frontend architecture, design system, and product presentation for the Aurall ecosystem.

🚀
The Aurall Website is built to communicate:

  🎧 Intelligent interaction recording
  🔐 Security-first architecture
  📊 Structured, actionable insights
  ⚡ High-performance, scalable infrastructure

🛠
Tech Stack

  Next.js – SSR, routing, and performance optimization
  React – Component-based UI architecture
  TypeScript – Type safety and maintainability
  Tailwind CSS – Utility-first styling

Built to present intelligence, security, and innovation — without overcomplicating the experience.
